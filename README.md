=====

ZSTD lossless compression algorithm bindings for node.js.

```bash
$ npm install zstd
```

## Usage

```js
var zstd = require('zstd');

var data = new Buffer(1000);
data.fill("A");

var compressed = zstd.compress(data);
var decompressed = zstd.decompress(compressed);
```
